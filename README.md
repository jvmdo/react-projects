# My React Projects

This repository is a hub for my React-based projects.

Each link redirects to a repository containing a descriptive detailed README, including app screencast and live preview.

## Projects

All project were built from either a Figma or a PNG design. They share React + Typescript.

### URL Shortening App

**Description:** responsive web page for creating shorter links using CleanURI API. Lots of CSS tricks in this one.

**Built with:** Base UI, Tanstack Query, TailwindCSS, Zod, Motion, Sonner

**Repository:** [jvmdo/url-shortening-app](https://github.com/jvmdo/url-shortening-app)

### IP Address Tracker

**Description:** a web application that looks up IP addresses and visualizes their approximate location on an interactive map.

**Built with:** Vite, Leaflet, Linaria, Tanstack Query, Maska, Zod

**Repository:** [jvmdo/ip-address-tracker](https://github.com/jvmdo/ip-address-tracker)

### Weather Now

**Description:** a responsive weather web app using OpenMeteo API. Look up for current weather, 7-day forecast, daily, hourly, details of any location in the planet. Switch between metric and imperial units.

**Built with:** Vite, CSS Modules, Tanstack Query/Pacer, Base UI, match-sorter, dayjs

**Repository:** [jvmdo/weather-now](https://github.com/jvmdo/weather-now)

### Animated Todo App

**Description:** responsive, animated todo list. Supports adding/marking/deleting todos, filtering (all/active/completed), light/dark themes, drag-and-drop reordering and animated transitions.

**Built with:** Vite, TailwindCSS, FormKit Drag and Drop, Motion, BaseUI

**Repository:** [jvmdo/animated-todo-app](https://github.com/jvmdo/animated-todo-app)

### Calculator Basic
